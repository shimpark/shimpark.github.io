블로그 구축기 1 (Hugo + github.io)
예전에는 문서화 작업에 그렇게 큰 신경을 두지 않았다. 프로젝트를 진행할 때 이것저것 삽질해보고 구글링하면서 문제를 해결하고선 이를 따로 정리해 두지 않고 완성해서 잘 동작시키는 것에만 의의를 두었다. 그러다보니 예전에 부딪혔던 문제를 다시 만나도 어떻게 해결했는지 기억이 안 나서 결국 같은 작업을 반복하는 경우가 생겼다. 이러한 일이 반복되자 내 작업 과정을 문서화해두는 작업이 필요함을 느꼈다. 그렇게 생각해보면 내가 그동안 구글링하면서 참고했던 블로그의 글들도 누군가의 작업 기록이었고, 나는 그들의 글을 보면서 내가 좀 덜 삽질할 수 있게 많은 도움을 받았다.
